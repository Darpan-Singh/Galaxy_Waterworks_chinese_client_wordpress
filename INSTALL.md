# йҠҖжІійҳІж°ҙ вҖ” WordPress Theme Installation Guide

## Folder to copy
Copy the entire folder:
```
wp-content/themes/galaxy-waterproof/
```
Into your WordPress installation:
```
/your-wordpress-root/wp-content/themes/
```

## Activate the theme
1. Log in to WordPress Admin вҶ’ Appearance вҶ’ Themes
2. Find "Galaxy Waterproof" and click **Activate**

## Set the homepage
1. Create a blank Page titled "йҰ–й Ғ"
2. Go to Settings вҶ’ Reading
3. Set "Your homepage displays" to **A static page**
4. Choose "йҰ–й Ғ" as the Homepage
